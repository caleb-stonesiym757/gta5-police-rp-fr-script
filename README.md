## Project Summary

GTA5 PoliceRP FR is meant to act as a setup guide and translation toolkit for GTA V police roleplay installs on Windows. The focus is on LSPDFR and the steps around getting that environment ready, with added help for applying French localization where it exists.

It also touches the practical pieces that often appear during installation, such as voice AI setup notes, Gemini-related guidance, RAGEPluginHook preparation, and keyboard/mouse input mapping. Rather than being a gameplay mod, it serves as a companion that helps users assemble and organize the components needed for a working police RP build.

- Guided installation flow for the complete setup
